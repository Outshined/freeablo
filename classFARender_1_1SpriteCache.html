Detailed Description

Multithread sprite cache.

FASpriteGroup is returned from the get() functions used in the game thread, which do not actually do image loading. get(size_t index) method is used to get a Render::SpriteGroup pointer in the render thread, actual image loading is done lazily here. The index value comes from FASpriteGroup.spriteCacheIndex
